# TRUNGFOOD - ECOMMERCE

## Yêu cầu

Trước khi chạy, đảm bảo máy của bạn đã cài đặt:

- **Node.js** (v20.19.3)
- **Docker Desktop** (cho Kafka)
- **pnpm**

## Hướng dẫn chạy dự án

### 1. Chạy và cài đặt dự án

```bash
# Tải code từ Github và giải nén nó
# Điều hướng đến thư mục gốc của project
cd trungfood

# Cài đặt phụ thuộc
pnpm install
```
